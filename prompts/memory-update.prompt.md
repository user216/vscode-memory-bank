# Update Memory Bank

Systematically review and update all Memory Bank files.

## Steps
1. Read ALL memory bank files in order:
   - projectbrief.md, productContext.md, systemPatterns.md, techContext.md
   - activeContext.md, progress.md
   - tasks/_index.md, decisions/_index.md
2. Analyze current codebase state vs documented state
3. Update activeContext.md with current focus and recent changes
4. Update progress.md with what's working, what remains, known issues
5. Update task statuses in tasks/_index.md
6. Add any new patterns discovered to systemPatterns.md
7. Report summary of all changes made
