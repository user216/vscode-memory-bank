# ADR-0008: Auto-commit memory-bank.db via git hooks and extension activation

## Context
The MCP server's SQLite DB contained imported items, FTS indexes, and cross-reference links that needed to be committed to git.

## Decision
**DEPRECATED**: SQLite was eliminated in [[ADR-0016]]. No database files exist to commit. Git hooks for DB staging are no longer needed.

Original decision: Implement 3-layer automatic approach (extension activation patches .gitignore, pre-commit hook stages DB, session-stop hook checkpoints WAL).

## Consequences
- This ADR is preserved for historical context only
- No database files, no WAL, no pre-commit hook for DB staging
- Git hooks simplified — no SQLite-specific logic
