# ADR-0009: Extract shared helpers into shared-utils.ts

## Context
Dead-code analysis identified 15 duplicated function definitions across 6 MCP tool files: `getMemoryBankPath` (6 copies), `slugify` (3 copies), `updateDecisionIndex` (3 copies), `getNextAdrId` (2 copies), plus structural analogs.

## Decision
Created `mcp/src/tools/shared-utils.ts` centralizing all duplicated helpers. Functions parameterized where structurally analogous. Status constants exported as `as const` tuples. Fixed `DecisionStatus` type to include "Rejected".

## Consequences
- Reduced from 15 duplicated function definitions to 1 shared module
- Bug fix: decision index now consistently uses correct default status
- All existing tests pass with no behavioral changes for callers
