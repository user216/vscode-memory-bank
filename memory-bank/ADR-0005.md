# ADR-0005: Agent Identity is Claude Agent Preview, Not Claude Code CLI

## Context
During development, the AI agent repeatedly misidentified itself as "Claude Code" (Anthropic's standalone CLI tool) and referenced Claude Code documentation, settings files, and behaviors. The actual runtime is the **Claude Agent Preview** running inside VS Code via the GitHub Copilot extension's Claude Agent SDK.

## Decision
Enforce strict agent identity at the instruction level (Layer 0). The agent must never identify as Claude Code CLI. Key differences documented in techContext and instructions.

| | Claude Agent Preview (this project) | Claude Code CLI |
|---|---|---|
| Runtime | VS Code + GitHub Copilot extension | Standalone terminal CLI |
| Billing | GitHub Copilot subscription | Anthropic subscription |
| MCP config | `.vscode/mcp.json` | `.mcp.json` or other locations |
| Settings | VS Code settings | `.claude/settings.json` |

## Consequences
- Identity enforcement rules added to Layer 0 instructions
- `techContext.md` documents the distinction with comparison tables
- Prevents future misidentification and incorrect advice
