# ADR-0006: MCP Config Location is .mcp.json at Project Root

## Context
The VS Code ecosystem has multiple possible locations for MCP server configuration. The agent repeatedly gave incorrect advice about which file the Claude Agent SDK reads.

## Decision
**DEPRECATED**: This decision was based on early Claude Agent SDK behavior. The SDK no longer uses `.mcp.json`. The extension now generates `.vscode/mcp.json` for GitHub Copilot. See [[ADR-0017]] and [[ADR-0018]] for current MCP configuration approach.

Original decision: The Claude Agent SDK reads MCP servers exclusively from `.mcp.json` at the project root.

## Consequences
- This ADR is preserved for historical context only
- Current MCP config: `.vscode/mcp.json` auto-generated by extension
- `.mcp.json` is no longer generated or used
