# ADR-0013: Distribute as Copilot Agent Plugin via Git submodule

## Context
VS Code Agent Plugins (Preview, March 2026) are a new distribution model — Git repos with plugin.json, skills, agents, hooks, and MCP configs. The memory-bank toolkit already has all the assets but distributes them only as a monorepo or VSIX.

## Decision
Create a separate Git repo (`user216/memory-bank-plugin`) containing all plugin assets and add it as a submodule at `copilot-plugin/`. This gives dual distribution: VSIX extension for the full VS Code UI experience, and Agent Plugin for zero-config Copilot agent integration.

## Consequences
- Plugin repo must be kept in sync when skills/agents/hooks/prompts change
- Dual distribution: VSIX for UI features, Agent Plugin for Copilot agent integration
- Plugin is skills + agents + hooks + prompts only (MCP delivered by extension — see [[ADR-0017]])
